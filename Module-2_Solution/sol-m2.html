Flexible Webpage !

Desktop Version

In desktop view, when width of the browser window is 992 px and above the page will display all three boxes side by side in one line. The width of each individual box is 33.33% the width of the browser window This is what you'll see when the browser window is opened in fullscreen mode.

Tablet Version

In the tablet view,when width of the browser window is anywhere from 768px to 991px, the first 2 boxed appear in the first row and are of equal size(50%). The 3rd box is in the second row and spans the entire row by itself. For a visual reference of this view,reduce the size of your broswer window to 991 pixels :).

Mobile View

In the mobile view, when width of the browser window is less than or equa to 767px, all three boxes appear in a separate line one below the other. The size of each box is 100% the width of the browser window. For a visual reference, reduce the size of your browser window to 767 pixels :)
